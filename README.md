# Vetur E

This is a branch of Vetur, adding a setting (vetur.extensions) to the original one. Used to turn on syntax services for vue files that have been changed to other extensions.

Please do not install and use Vetur E and Vetur at the same time.

## Features

- Syntax-highlighting
- Snippet
- Emmet
- Linting / Error Checking
- Formatting
- Auto Completion
- Debugging

## Quick Start

- Install [Vetur](https://marketplace.visualstudio.com/items?itemName=octref.vetur).
- Try it with [Veturpack](https://github.com/octref/veturpack), a pre-configured Vue boilerplate based on [Vuepack](https://github.com/egoist/vuepack).
- Refer to [setup](https://vuejs.github.io/vetur/setup.html) page for setting up.
- Refer to each feature's own page for setting up specific features.
